# Идеи

- Быстрый запуск медитации с главного экрана
- Режимы: сон, расслабление, дыхание, фокус
- Минималистичный glossy Apple-style UI
- Дыхательный круг как главный элемент управления
- Прокручиваемый sound picker вместо плиток
- Основной звук через MP3-фоны, частоты через AudioTrack как мягкий слой
- Мужской voice-over для guided meditation
- Первые 3 голосовые сессии как быстрый MVP для проверки качества
- В будущем расширить библиотеку до 20-50 аудио
- Детская сказка-медитация про маленького ёжика и самую тихую звезду
